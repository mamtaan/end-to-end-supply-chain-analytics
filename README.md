 # 🚚 Supply Chain Delivery Performance Analysis & Delay Prediction
📌 Project Overview

This project analyzes a global e-commerce supply chain dataset to identify the key causes of delivery delays and understand their impact on business profitability and operational efficiency.

The project combines:

Data Cleaning & Feature Engineering
Exploratory Data Analysis (EDA)
Business KPI Analysis
Root Cause & Bottleneck Detection
Machine Learning Prediction
Power BI Dashboarding

The final goal is to help businesses:

Reduce late deliveries
Improve shipping performance
Optimize operational efficiency
Increase profitability through data-driven insights
📊 Dataset Information

The dataset contains:

Customer information
Shipping details
Product categories
Order status
Delivery performance
Profitability metrics
Regional and time-based operational data
Dataset Size
Rows: 172,765
Columns: 20 (after cleaning)

🔍 Key Business Problems
High percentage of late deliveries
Operational inefficiencies across shipping modes and regions
Profit reduction due to delayed shipments
Need for predictive delivery risk analysis
📈 Exploratory Data Analysis

The analysis includes:

Delivery delay distribution
Profitability analysis
Regional bottleneck detection
Shipping mode comparison
Department-wise delay analysis
Time-based trend analysis
Root cause analysis
📌 Key Insights
Over 54% of orders were delayed
First Class shipping showed the highest delay percentage
Central Africa and Southeast Asia experienced the highest delivery delays
Delays negatively impacted overall profitability
Peak delays occurred during August, September, and December
🤖 Machine Learning Model

A machine learning model was built to predict the risk of late delivery.

Models Compared
Logistic Regression
Decision Tree
Random Forest
XGBoost
Final Selected Model

✅ Random Forest Classifier

Model Performance
Accuracy: 73.8%
Precision: 78.5%
Recall: 74.6%
F1 Score: 76.5%
